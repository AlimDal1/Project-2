# Advanced ETL Data Pipeline

## Overview
This project implements a scalable ETL pipeline using Python with modular architecture, logging, validation, and database integration.

## Features
- Modular ETL design (Extract, Transform, Load)
- Data validation layer
- Logging system for monitoring
- SQLite database integration
- Config-driven pipeline

## Tech Stack
- Python (Pandas)
- SQLite
- YAML (configuration)
- Logging

## Use Case
Designed for handling humanitarian or operational datasets, ensuring data quality, consistency, and reliability.
